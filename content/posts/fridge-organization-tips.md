## 결론부터

냉장고 문을 연다. 뒤쪽에 뭔가 있는 건 아는데 손이 안 닿는다. 앞에 있는 것들을 하나씩 빼다 보면 유통기한 한 달 지난 두부가 나온다. 작년에 산 냉동 만두도 발견된다. 이러면서 "냉장고가 작아서 그래"라고 생각하지만, 실제로는 **정리 방법**의 문제입니다.

냉장고 수납의 핵심은 **"열었을 때 한눈에 전부 보이는가"**입니다.

- **선입선출**: 새 식재료는 뒤, 오래된 건 앞
- **투명 용기**: 내용물이 보여야 잊어버리지 않음
- **칸별 역할 배정**: 유제품 칸, 반찬 칸, 음료 칸 고정

## 냉장실 칸별 최적 배치

냉장고 온도는 칸마다 다릅니다. 위쪽이 덜 차고, 아래쪽이 더 차갑습니다.

| 위치 | 온도 | 넣을 것 |
|---|---|---|
| 최상단 | 3~5℃ | 음료, 조미료, 즉석 먹을 것 |
| 중간 칸 | 2~4℃ | 반찬, 두부, 계란 |
| 하단 칸 | 1~3℃ | 생선, 육류 (드립 방지 트레이 위) |
| 도어 포켓 | 5~7℃ | 소스, 잼, 물, 자주 꺼내는 것 |
| 채소칸 | 5~7℃ (습도 높음) | 채소, 과일 |

### 왜 육류는 아래에?

육류에서 핏물(드립)이 흘러내리면 아래 식재료를 오염시킵니다. 최하단에 놓고 트레이 위에 올려두면 안전합니다.


## 수납 용기 — 투명이 답

냉장고 수납의 90%는 **용기 선택**으로 결정됩니다.

### 추천 수납 용기

| 용기 | 용도 | 가격대 |
|---|---|---|
| 투명 밀폐용기 (사각) | 반찬, 소분 | 3~5천/개 |
| 냉장고 정리 트레이 | 소스·양념 묶음 | 3~8천/개 |
| 에그트레이 | 계란 정리 | 3~5천 |
| 캔음료 디스펜서 | 캔 굴러다니는 것 방지 | 5~8천 |
| 냉동실 칸막이 | 냉동 식품 세워서 보관 | 3~5천 |

### 용기 선택 원칙

1. **사각형** > 원형 (공간 효율 30% 차이)
2. **투명** > 불투명 (내용물 안 보이면 잊어버림)
3. **같은 브랜드** 통일 (스태킹 가능)
4. **전자레인지 가능** 소재 (옮겨담기 귀찮음 방지)

## 냉동실 — 세워서 보관이 핵심

냉동실에 만두 위에 고기 위에 밥 위에 또 만두. 아래에 뭐가 있는지 모르는 "지층 구조"가 됩니다.

### 해결법: 세워서 보관

1. 식재료를 **지퍼백**에 평평하게 담기
2. **세로로 세워서** 파일처럼 꽂기
3. 칸막이로 영역 구분 (육류/생선/밥/기타)

이렇게 하면 서랍처럼 한눈에 전체가 보이고, 원하는 것만 쏙 꺼낼 수 있습니다.

### 냉동실 수납 순서

| 구역 | 내용 |
|---|---|
| 왼쪽 | 육류·생선 (날것) |
| 가운데 | 밥·국·소분 반찬 |
| 오른쪽 | 만두·냉동식품·빵 |
| 도어 | 아이스크림·얼음팩 |


## 식재료 유통기한 관리

냉장고 정리의 진짜 목표는 **버리는 음식을 줄이는 것**입니다.

### 실천 가능한 방법

1. **마스킹 테이프 + 네임펜**: 용기에 날짜 적기 (개봉일)
2. **주 1회 냉장고 점검**: 장보기 전에 한 번 훑기
3. **선입선출**: 새로 산 건 뒤로, 기존 건 앞으로
4. **냉동 보관 기한** 숙지:

| 식재료 | 냉동 보관 기한 |
|---|---|
| 소고기/돼지고기 | 3~6개월 |
| 닭고기 | 2~3개월 |
| 생선 | 1~3개월 |
| 밥 | 1개월 |
| 빵 | 2~3개월 |


## 냉장고 냄새 잡기

정리를 해도 냄새가 나면 기분이 안 좋습니다.

- **탈취제**: 냉장고 전용 숯 탈취제 (2~3개월마다 교체)
- **커피찌꺼기**: 건조 후 거즈에 싸서 넣기 (무료)
- **원인 제거**: 대부분 냄새는 덮개 없이 보관한 반찬에서 남 → 밀폐용기가 근본 해결

## 자주 묻는 질문

**Q1. 냉장고가 작은데 수납이 될까요?**
A. 작은 냉장고일수록 수납 기술이 중요합니다. 용기를 통일하고 세로 보관하면 체감 용량이 30~50% 늘어납니다. "냉장고가 작은 게 아니라 정리가 안 된 것"인 경우가 대부분입니다.

**Q2. 수납 용기 한 번에 다 사야 하나요?**
A. 아닙니다. 가장 많이 쓰는 반찬 용기 4~6개 + 트레이 2개로 시작하세요. 사용하면서 필요한 크기를 파악한 후 추가 구매가 낭비를 줄입니다.

**Q3. 채소는 씻어서 넣어야 하나요?**
A. 대부분의 채소는 **씻지 않고** 보관하는 게 더 오래 갑니다. 물기가 곰팡이를 부릅니다. 키친타월로 감싸고 지퍼백에 넣으세요. 먹기 직전에 씻는 게 정답입니다.
